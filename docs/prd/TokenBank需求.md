# 项目
TokenBank - 代币银行 DApp。
# 核心功能
标准 ERC20 代币存取功能。
# 特性：
1、标准 ERC20 代币 (MyToken)
2、基础存取合约 (TokenBank)
3、Approve/TransferFrom 模式
4、Next.js 16 前端
5、实时余额显示
# 技术栈：
1、前端框架：Next.js
2、类型检查：TypeScript
3、样式框架：Tailwind CSS 4
4、钱包连接库：Wagmi v2 + Viem v2
5、状态管理库：React Query
6、RainbowKit (钱包连接)
7、React Query (状态管理)

# 功能描述：
1. 连接钱包
2. 钱包连接后显示当前 Token 的余额，存款余额。
3. 存款(点击按钮存款)到 TokenBank。存款后显示用户存款金额。
4. 支持用户取款(点击按钮取款)。
5. 授权后显示当前授权额度。
